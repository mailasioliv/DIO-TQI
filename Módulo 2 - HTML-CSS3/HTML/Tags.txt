<!DOCTYPE html> = tipo de documento

<html> </html> = tag que contém todos elementos HTML

<head></head> = informações necessárias para o buscador e navegador
<meta> = um dos tipos de metainformações que vai dentro de <head> ex. charset que é o incode dos caractéres
<title></title> = Título na aba do navegdor

<body></body> = conteúdo da página

<section> = seção genérica de conteúdos
<header> = cabeçalho da página
<article> = conteúdo relevante
<aside> = conteúdo relacionado ao conteúdo principal da página
<footer> = rodapé da  página ou de um article/section
<h1>-<h6> = títulos
<p></p> = parágrafos, textos mais densos, imagens, códigos, vídeos...
<a></a> = âncora: href, ou hiperlink apontando para telefone ou e-mail
<a href="link">LinkedIn</a> = nomear link
<a target="_blank">Link</a> = abrir link em nova página.
<img> = inserir imagem
src= fonte/caminho da imagem
alt= texto/descrição da foto quando não é carregada
<ul> = lista em que a ordem dos itens não é importante
<ol> = lista em que a ordem dos itens é importante
<li> = itens da lista
